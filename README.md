# FoodBazar

2. Description / Purpose

FoodBazar is a food delivery platform built with React, Node.js, and MongoDB. Users can browse meals, place orders, submit reviews, and favorite their favorite meals. Admins and chefs have separate dashboards for managing the platform.

3. Live Demo URL



## Live Demo
[https://localchefbazar-roan.vercel.app](https://localchefbazar-roan.vercel.app)

4. Features

## Features
- User authentication with Firebase (email/password)
- Role-based dashboards: Admin, Chef, User
- Browse meals, sort meals by price
- Meal details with reviews and favorite button
- Place orders and pay with Stripe
- Admin can manage users, requests, and view platform statistics
- Responsive design for mobile and desktop

8. Usage Instructions

## Usage

1. Register as a user.
2. Browse meals on the Home or Meals page.
3. Click "See Details" to view meal info.
4. Add reviews or favorite meals.
5. Place an order and pay with Stripe.
6. Chefs can add/update meals.
7. Admins can manage users, requests, and view platform statistics.
